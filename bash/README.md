# Bash Scripts

Small Linux utility scripts and operational helpers.

## Monitoring & Validation

- **check_cert_expiry.sh**  
    Lightweight helper to check TLS certificate expiration for endpoints and warn when certificates are approaching expiry.

 ## Diagnostics

- **endpoint_diagnostics.sh**  
  Troubleshooting utility that checks DNS resolution, TCP connectivity, TLS certificate status, HTTP response code, and latency for service endpoints.
